# 愿景

## 项目目标

在 Apple Silicon (M4 Pro) 上用 Python/PyTorch 模拟 NVIDIA DLSS 4.5 的核心算法管线，理解其工作原理，产出可运行的离线模拟 demo。

## 核心理念

**学习优先，模拟其次**。本项目的首要目标是深入理解 DLSS 4.5 的技术原理，而非实现一个可替代 DLSS 的实时方案。模拟的作用是"通过实现来验证理解"。

## 覆盖范围

DLSS 4.5 三大核心技术栈：

| 组件 | 学习目标 | 模拟范围 |
|------|---------|---------|
| 超分辨率上采样 | Vision Transformer 架构如何替代 CNN | ✅ 离线推理单帧/序列 |
| 动态多帧生成 | 光流法如何从 1 帧插出多帧 | ✅ 离线处理短视频 |
| 光线重建 | AI 降噪替代传统多降噪器 | ⚠️ 简化版（复杂度高） |

## 工作流（4 Phase）

```
Phase 1: 原理研究       →  理解每个模块的论文和实现
Phase 2: 环境搭建       →  M4 Pro 上跑通开源 demo  
Phase 3: 模块化模拟     →  分别实现超分/帧生成/降噪
Phase 4: 集成与评估     →  串联管线 + 量化指标
```

## 核心原则

1. **正确性第一**：理解错了比跑得慢更糟糕
2. **模块化**：每个子系统独立可测试
3. **可复现**：所有实验记录参数、种子、环境
4. **素材卡片化**：研究过程中收集的参考资料用统一模板记录
