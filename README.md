# DLSS 4.5 模拟项目

在 Apple Silicon (M4 Pro) 上模拟 NVIDIA DLSS 4.5 核心算法管线。

## ⚠️ 开展工作前必读

每次开展本项目的工作之前，必须先了解 **PM/** 目录下的内容：

```
PM/
├── 05-需求记录.md        ← 🔴 最重要：所有需求及完成状态
├── 01-愿景.md            ← 项目目标、工作流、覆盖范围
├── 02-项目状态.md        ← 当前各 Phase 进度
├── 03-管理原则与方法.md   ← 怎么干、工具链
└── 04-决策记录.md        ← 已做决策，别重复讨论
```

**阅读顺序：需求记录 → 状态 → 愿景 → 原则 → 决策**

## 项目结构

```
dlss-simulation/
├── PM/                   ← 项目管理（必读）
├── research/             ← Phase 1：原理研究笔记、素材卡片
├── src/                  ← Phase 2-3：模拟代码（按模块）
│   ├── super_resolution/
│   ├── frame_generation/
│   └── denoising/
├── experiments/          ← Phase 3-4：实验脚本
├── data/                 ← 测试素材（不提交 git）
└── references/           ← 论文、博客等外部资料
```
